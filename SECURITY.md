
# Seguridad del contenido

## Repositorio privado

El repositorio contiene temario, bancos, procedimientos y materiales internos. Debe permanecer privado y con acceso limitado a las personas que trabajan en Academia En Vigor.

## Incidencias

Si se incorpora por error un PDF, DOCX, escaneo, audio, vídeo, presentación o archivo de terceros:

1. detener cualquier publicación o distribución;
2. eliminarlo del árbol de trabajo;
3. comprobar si llegó al historial de Git;
4. si llegó al historial, reescribirlo antes de continuar;
5. registrar la incidencia y revisar los permisos del repositorio.

Los secretos, contraseñas, claves API y datos personales nunca se guardan en el repositorio.
