# Evaluaciones · Tema 3

`plan.json` es la configuración canónica. El generador crea en `build/`:

- 40 tests de cobertura por bloques;
- 18 tests por partes;
- 3 tests finales de 25, 50 y 100 preguntas.

Las 738 preguntas aparecen exactamente una vez en el conjunto de tests de cobertura. Las salidas se regeneran y no se suben al repositorio.

```bash
python scripts/generar_evaluaciones_tema_03.py
```
