# Imágenes integradas · Tema 3

Esta carpeta contiene únicamente imágenes, esquemas y gráficos que aparecen dentro de El Parte o El Atestado.

## Nombres recomendados

```text
t03-01-corona-refrendo.png
t03-02-funciones-del-rey.png
t03-03-sucesion-regencia-tutela.png
t03-04-cortes-generales.png
```

Reglas:

- usar `t03-` al principio;
- numeración de dos cifras;
- nombres en minúsculas y con guiones;
- preferir PNG o SVG;
- registrar cada archivo en `manifest.json`;
- indicar en qué documento y sección se utiliza.

Las infografías completas que el alumno abre por separado no van aquí; van en `materiales-didacticos/.../infografias/`.
