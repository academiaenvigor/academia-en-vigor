
# Política de fuentes y propiedad intelectual

## Permitido

- enlaces y metadatos de fuentes oficiales;
- contenido redactado por Academia En Vigor;
- assets propios o con permiso documentado;
- registros normalizados de preguntas históricas sin adjuntar ediciones de terceros.

## Prohibido

- PDF, DOCX, escaneos o capturas de otras academias;
- marcas de agua, logotipos o maquetaciones ajenas;
- retroalimentaciones y razonamientos copiados;
- recursos sin autoría, licencia o permiso claro;
- archivos privados usados únicamente para comprobación.

## Archivo privado de comprobación

Los originales que se necesiten para contrastar información se guardan fuera de Git y no se comparten con alumnos. El repositorio solo conserva una referencia interna no reversible, nunca el archivo.
