
# Arquitectura del repositorio

El repositorio separa cinco tipos de contenido:

1. **Conocimiento:** la fuente maestra y los hechos atómicos.
2. **Temas derivados:** El Parte y El Atestado.
3. **Entrenamiento propio:** preguntas y planes de evaluación.
4. **Evidencia histórica:** exámenes oficiales normalizados y sus índices por tema.
5. **Materiales:** assets insertados y recursos independientes.

La fuente maestra se modifica primero. Después se regeneran derivados, cobertura, preguntas, tests y materiales. Los exámenes históricos nunca sustituyen a una fuente normativa.
