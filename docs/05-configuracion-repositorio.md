
# Configuración recomendada de GitHub

1. Cambiar la visibilidad del repositorio a **privado** antes de incorporar el banco histórico.
2. Proteger la rama `main` y exigir una solicitud de incorporación.
3. Exigir que la acción `Validar contenido VIGOR` termine correctamente.
4. Desactivar la inserción directa de archivos grandes.
5. Conceder acceso únicamente a colaboradores necesarios.
6. Mantener una copia de seguridad periódica fuera de GitHub.

La etiqueta `not_published` dentro de un JSON no protege el contenido si el repositorio es público.
