
# Publicación y copias de seguridad

El repositorio editorial no es la plataforma de alumnos. La web o aplicación consume únicamente contenidos aprobados y exportados.

## Publicación

1. cerrar la versión del tema;
2. validar fuente, banco, evaluaciones y materiales;
3. generar una exportación para alumnos;
4. publicar la exportación, no el repositorio completo.

## Copia de seguridad

Conservar periódicamente:

- una copia cifrada del repositorio;
- una copia de los archivos externos propios;
- un inventario de enlaces y permisos;
- una exportación de la base publicada.
