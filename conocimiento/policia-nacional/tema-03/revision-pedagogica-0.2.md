# Revisión pedagógica 0.2 · Tema 3

## Motivo

La versión anterior del Atestado era jurídicamente exhaustiva, pero presentaba demasiados hechos consecutivos en forma de lista. Este formato dificultaba la lectura sostenida y podía provocar que el alumno abandonara el bloque tras pocos puntos.

## Cambios

- Se mantienen los 25 bloques, los 470 hechos atómicos, las fuentes y las imágenes.
- Se conserva El Parte como versión rápida.
- El Atestado adopta una estructura narrativa:
  - explicación previa de cada bloque;
  - desarrollo en párrafos agrupados por artículos;
  - ejemplos mentales;
  - estrategia de estudio;
  - cierre con los conceptos de examen.
- Los identificadores `FACT` permanecen junto a cada afirmación para conservar la trazabilidad con el banco de 738 preguntas.
- No se ha modificado el contenido normativo ni la atribución de preguntas oficiales.

## Resultado esperado

El alumno puede comprender primero la lógica del bloque, leer después el detalle jurídico y utilizar los elementos destacados como puntos de recuperación, sin enfrentarse a una sucesión continua de viñetas.
